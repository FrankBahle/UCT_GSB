## 🚀 How to Use

### For Instructors
1. Open `index.html` in a web browser
2. Navigate through each day using the navigation bar
3. Access all resources from the Resources dropdown
4. Students can track their progress automatically
5. Print any page for offline reference

### For Students
1. Start with Day 1
2. Complete assessments when prompted
3. Take quizzes to test knowledge
4. Review case studies for real-world examples
5. Progress is saved automatically
